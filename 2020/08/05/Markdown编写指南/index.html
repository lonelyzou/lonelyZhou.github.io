什么是Markdown

Markdown是一种轻量级标记语言，通过纯文本的方式编写文档，然后再通过工具转化成HTML，PDF，图片等格式，实现Write Once，Read Everywhere的效果。下面直接开始指南

支持HTML元素
不在 Markdown 涵盖范围之内的标签，都可以直接在文档里面用 HTML 撰写。 目前支持的 >HTML 元素有：<kbd> <b> <i> <em> <sup> <sub> <br>等。
